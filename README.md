# Gitem - Simple Github App

## Uses the Github developer API to search for repositories and users, as well as user details

## Tech stack
The app was built using:
- Android SDK
- Jetpack Compose
- Kotlin
- MVVM Architecture
- Material 3 design lib
- Retrofit 2 & OkHTTP
- Gson
- JUnit and Mockito
- Paging 3 Lib
